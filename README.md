 Book Management Script
This is a simple Python project that uses **Classes and Objects** to manage book data.

 Features
- **Data Encapsulation**: Stores ID, Title, Author, and Price.
- **Discount Logic**: A method to automatically calculate and apply percentage-based discounts.
- **Formatted Output**: Displays book details in a clean, readable format.

 How to Run
1. Ensure you have Python installed.
2. Run `python your_filename.py` in your terminal.
